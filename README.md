# miniproject
------------
## 1. mini project 주제
------------

  학관 메뉴 주문/관리 프로그램 만들기



## 2. mini project 에 대한 소개
주문과 메뉴 관리 두개로 나누어 메뉴 관리 버전의 경우 일주일 동안의 학관 메뉴를 파일을 통해 불러오고 CRUD기능,  요일 별 검색이 기능 가능.


주문 버전의 경우 crud가 가능한 프로그램이다.
